# Dependencies

* Unix-like OS
* R
* Ruby
* SQLite

## Ruby libraries
* sequel
* trollop 
* CSV

To install:
```bash
> gem install sequel trollop CSV
```

## R libraries
* sqldf

To install (from the R console):
```R
install.packages("sqldf")
```
